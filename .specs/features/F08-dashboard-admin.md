# F-08 — Dashboard Admin

**Status:** COMPLETO

---

## Contexto

O dashboard é a tela principal do administrador. Oferece uma visão geral do desempenho da rede para que o gestor tome decisões informadas rapidamente.

---

## Histórias de Usuário

### US-01 — Ver KPIs da rede
**Como** admin,
**quero** ver os principais indicadores da rede em uma tela,
**para que** eu tenha uma visão rápida do estado atual do programa de treinamento.

**Critérios de aceitação:**
- [ ] Exibe total de lojas cadastradas
- [ ] Exibe total de usuários ativos
- [ ] Exibe total de vídeos disponíveis
- [ ] Exibe progresso médio de todos os colaboradores (%)

### US-02 — Ver ranking de lojas por desempenho
**Como** admin,
**quero** ver quais lojas têm o melhor desempenho em treinamento,
**para que** eu identifique lojas que precisam de atenção ou que servem de referência.

**Critérios de aceitação:**
- [ ] Lista de lojas ordenadas por score de progresso médio (maior → menor)
- [ ] Cada loja mostra: nome, número de colaboradores e score
- [ ] Visual diferencia as lojas líderes das que estão abaixo da média
